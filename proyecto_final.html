Generar un traductor de código

El objetivo del proyecto es integrar lo aprendido hasta el momento sobre las fases de un compilador, en particular el analizador lexico y sintactico, se debe tomar como entrada un codigo fuente que funcione, en este caso un codigo de Java y traducirlo a un lenguaje diferente respetando las reglas de sintaxis o lexicas del lenguaje al que se traducira, se podra elegir entre Python o R como lenguaje destino, si desean un lenguaje diferente pueden sugerirlo.

Pueden encontrar teoria en el Capitulo 5 del libro Compiladores principios, tecnicas y herramientas.

Posibles pasos para el proyecto, pueden variar dependiendo de la logica de cada equipo:

  1. Crear una función que lea un archvo txt y devuelva una lista con cada linea.
  2. Crar una lista de tokens o porciones de codigo equivalentes al lenguaje al que se traducira similar a la creada en el proyecto 2.

  4. Intercambiar cada token por el otro validando la sintaxis
